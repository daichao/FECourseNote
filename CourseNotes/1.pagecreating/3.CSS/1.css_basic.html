基本结构

div p, #id:first-letter {
	color: white;
	background-color: red;
}

以上代码逐级分析如下¹:

CSS声明(declaration)
color:white; 是一个CSS声明。其中color是属性，white是属性值，两者一起构成一个名值对，以英文冒号:分隔，以英文分号结束;。background-color: red;亦是如此。
CSS声明块(declaration block)
大括号{和}包围的部分，称之为声明块。声明块中可包含多个声明
选择器集合(group of selectors)
div p, #id:first-letter是一个选择器集合。具体内容在选择器中详细讲述。
CSS规则集(ruleset)
一对选择器(集合)和声明块，构成一个CSS规则集，简称CSS规则(rule)
CSS语句(statements)
规则和部分at规则，可以作为语句。具体内容在at规则中详细讲述。

属性值定义语法

符号² 名称 描述 示例
组合符号(组合优先级往后递减)
  并置 各部分必须出现且按顺序出现 solid <length>
&& “与”组合符 各部分必须出现，但可以不按顺序 <length> && <string>
|| “或”组合符 各部分至少出现一个，可以不按顺序 <'border-image-outset'> || <'border-image-slice'>
| “互斥”组合符 各部分恰好出现一个 smaller | small | normal | big | bigger
[ ] 方括号 强调优先级 bold [ thin && <length> ]
数量符号
  无数量符号 恰好一次 solid
* 星号 零次、一次或多次 bold smaller*
+ 加号 一次或多次 bold smaller+
? 问号 零次或一次（即可选） bold smaller?
{A,B} 大括号 至少A次，至多B次 bold smaller{1,3}
# 井号 一次或多次，但多次出现必须以逗号分隔 bold smaller#

样式引入方式

内联样式
将样式直接写在元素的style属性中。如： 
<span style="font-size:18px; color:red;">some contents</span>
内部样式表
在head元素里的最后部分写入style元素，其中直接写入CSS规则集。如： 
<head>
	<style type="text/css" media="screen">
		span {
			font-style: 18px;
			color: red;
		}
	</style>
</head>
外部样式表
在head元素里的link元素引入外部css文件。如： 
<link rel="stylesheet" href="/default.css">
