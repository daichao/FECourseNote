背景色|background-color

Name: background-color
Value: <color>
Initial: transparent
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: the computed color
Animatable: as color

这个属性用于设置背景颜色。

rgb(255, 0, 0) ，或者 rgba(255, 0, 0, 0.1) 设置透明度

#ff0000或者简写的#f00

也可以用颜色名

背景图|background-image

Name: background-image
Value: <bg-image> [ , <bg-image> ]*
Initial: none
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: as specified, but with URIs made absolute
Animatable: no 

<bg-image> = <image> | none
<image> = <url> | <image-list> | <element-reference>  | <gradient>
<gradient> = [ <linear-gradient> | <radial-gradient> | 
                        <repeating-linear-gradient> | <repeating-radial-gradient> ]

url()，参数绝对地址，相对地址，相对根路径，双引号，单引号，甚至不加引号。

instance

多个背景图片或背景颜色时，越出现在逗号左边，图片越在前面。

背景图平铺|background-repeat

Name: background-repeat
Value: <repeat-style> [ , <repeat-style> ]*
Initial: repeat
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: A list, each item consisting of: two keywords, one per dimension
Animatable: no 

<repeat-style> = repeat-x | repeat-y | [repeat | space | round | no-repeat]{1,2}

repeat-x相当于repeat-x no-repeat，沿x轴平铺图片。

repeat-y相当于no-repeat repeat-y，沿y轴平铺图片。

space相当于space space，在图片之间填充空间以适应边框。

round相当于round round，适当缩放，紧密排满边框。

no-repeat相当于no-repeat no-repeat，不平铺。

可以写两个值，如果有两个值，则前面一个是X轴的，后面一个是Y轴的，中间空格。也可以叠加多个图片的平铺方式，像背景图片用逗号隔开。

背景定位|background-position

Name: background-position
Value: <position> [ , <position> ]*
Initial: 0% 0%
Applies to: all elements
Inherited: no
Percentages: refer to size of background positioning area minus size of background image; see text
Media: visual
Computed value: A list, each item consisting of: a pair of offsets (horizontal and vertical) from the top left origin each given as a combination of an absolute length and a percentage
Animatable: as repeatable list of simple list of length, percentage, or calc 

<position> = [ 
                         [ left | center | right | top | bottom | <percentage> | <length> ] 
                        | 
                         [ left | center | right | <percentage> | <length> ]
                         [ top | center | bottom | <percentage> | <length> ] 
                        | 
                         [ center | [ left | right ] [ <percentage> | <length> ]? ] && 
                         [ center | [ top | bottom ] [ <percentage> | <length> ]? ] 
                        ]

这个属性用于定位背景图片的放置。

使用一个值，另一个值默认是center，不一定是水平或垂直。使用时要注意与TRBL补全的方式进行区分。

使用两个。10px 20px，x轴从左往右偏移10px，y轴从上往下偏移20px。 百分比20% 50%，图片的20%和50%这个交汇点与容器的20%和50%的交汇点的重合。 位置名称，right center，没有顺序限制，表示右边中间位置。

使用四个值。right 10px top 20px，距离右边10px，距离上边20px。这个例子参照物是由right和top决定，而上面两种情况则是以容器的左上角为原点，上边为x轴线，左边为y轴线。

精灵图/雪碧图/sprite，使用背景定位实现：

background-image:url(sprite.png);
background-repeat:no-repeat;
background-position:0 -100px;

背景绘制区域/裁剪|background-clip

Name: background-clip
Value: <box> [ , <box> ]*
Initial: border-box
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: as specified
Animatable: no 

<box> = border-box | padding-box | content-box

这个属性用于确定背景的绘制区域。

border-box，默认，截取边框boder外边界以内的区域，并显示。

test content

padding-box，截取边框border内边界以内的区域，并显示。

test content

content-box，截取content文本区域，并显示。

test content

注意：根元素有不同的区域绘制方式，因此background-clip对html根元素无效，看这里。默认情况下，背景总是绘制在边框border之后的，但是不会超出边框。

背景填充开始位置|background-origin

Name: background-origin
Value: <box> [ , <box> ]*
Initial: padding-box
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: as specified
Animatable: no 

<box> = border-box | padding-box | content-box

这个属性用于定位背景的从哪个区域开始平铺。

padding-box，默认，从边框border内边界以内的左上角开始开始平铺。

test content

border-box，从边框border外边界以内的左上角开始平铺。

test content

content-box，从content文本区域左上角开始平铺。

test content

注意：当 background-clip:padding-box; background-origin:borde-box; background-position:top left; （都是初始值）且元素的边框不是0时，左上角的背景将被截掉。

背景缩放|background-size

Name: background-size
Value: <bg-size> [ , <bg-size> ]*
Initial: auto
Applies to: all elements
Inherited: no
Percentages: see text
Media: visual
Computed value: as specified, but with lengths made absolute and omitted ‘auto’ keywords filled in
Animatable: as repeatable list of simple list of length, percentage, or calc (This means keyword values are not animatable.) 

<bg-size> = [ <length> | <percentage> | auto ]{1,2} | cover | contain

cover，图片宽高都不能小于容器宽高，图片尽可能小。即图片宽高包含容器宽高。

contain，图片宽高都不能超过容器宽高，且图片尽可能大。即容器宽高包含图片宽高。

背景|background

Name: background
Value: [ <bg-layer> , ]* <final-bg-layer>
Initial: see individual properties
Applies to: all elements
Inherited: no
Percentages: see individual properties
Media: visual
Computed value: see individual properties
Animatable: see individual properties 

<bg-layer> = <bg-image> || <position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box> || <box>
<final-bg-layer> = <bg-image> || <position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box> || <box> || <'background-color'>

注意：背景颜色只允许放在最下面一层的背景叠层里，即放在最右边的逗号后面。如果对body元素设置背景，那么这个背景会向上传递到根元素html，所以给html根元素或者body元素设置背景，都会让整个视窗都有了这个背景。 <position> 和 <bg-size> 之间一定要有 / 。语法最后两个 <box> 是指的 background-clip 和 background-origin ，其中第一个是 background-origin ，第二个是 background-clip 。

线性渐变|linear-gradient()

<linear-gradient> = linear-gradient(
                                [ [ <angle> | to <side-or-corner> ] ,]? 
                                <color-stop>[, <color-stop>]+
                                )
<side-or-corner> = [left | right] || [top | bottom]
<color-stop> = <color> [ <percentage> | <length> ]?

关键字默认是从下往上 to bottom 渐变的，角度 0deg 默认是从下往上的。第一个值是确定渐变的方向（可选），后面的值都是用以颜色和长度值或百分比（可选）为一个整体，用逗号隔开多个整体。

linear-gradient(yellow, blue);
linear-gradient(to bottom, yellow, blue);
linear-gradient(180deg, yellow, blue);
linear-gradient(to top, yellow, blue);
linear-gradient(to bottom, yellow 0%, blue 100%);
linear-gradient(135deg, yellow, blue);
linear-gradient(-45deg, yellow, blue);
linear-gradient(yellow, blue 20%, #0f0);
linear-gradient(to top right, red, white, blue);

径向渐变|radial-gradient()

<radial-gradient> = radial-gradient(
                                      [ [ <shape> || <size> ] [ at <position> ]? , |
                                        at <position>, 
                                      ]?
                                      <color-stop> [ , <color-stop> ]+
                                    )
<shape> = circle || ellipse
<size> = <extent-keyword> || <length>
<size> for ellipse = [ <length> | <percentage> ]{2}
<position>
<extent-keyword> = closest-corner | closest-side | farthest-corner | farthest-side
<color-stop> = <color> [ <percentage> | <length> ]?

此方法控制图像从中心点位置往外渐变。 <shape> 默认 circle 。 <size> 默认 farthest-corner。 <position> 默认在容器正中间 <center> 。如果 <shape> 是 <ellipse> 则 <size> 要取x轴和y轴两个值。 

radial-gradient(yellow, green)
radial-gradient(ellipse at center, yellow 0%, green 100%)
radial-gradient(farthest-corner at 50% 50%, yellow, green)
radial-gradient(circle, yellow, green)
radial-gradient(red, yellow, green)
radial-gradient(farthest-side at left bottom, red, yellow 50px, green)
radial-gradient(closest-side at 20px 30px, red, yellow, green)
radial-gradient(20px 30px at 20px 30px, red, yellow, green)
radial-gradient(closest-side circle at 20px 30px, red, yellow, green)
radial-gradient(20px 20px at 20px 30px, red, yellow, green);

循环渐变|repeating-linear-gradient() and repeating-radial-gradient()

循环渐变语法上对应线性渐变和径向渐变。

repeating-linear-gradient(red, blue 20px, red 40px)
repeating-radial-gradient(red, blue 20px, red 40px)
repeating-radial-gradient(circle closest-side at 20px 30px, red, yellow, green 100%, yellow 150%, red 200%);
