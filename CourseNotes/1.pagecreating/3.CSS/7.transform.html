变形|transform

Name: transform
Value: none | <transform-list>
Initial: none
Applies to: transformable elements
Inherited: no
Percentages: refer to the size of reference box
Media: visual
Computed value: As specified, but with relative lengths converted into absolute lengths.
Animatable: as transform

设置非none的属性值，都会产生一个堆叠内容（一个有3D效果的子元素）和一个容器块（这个元素本身作为一个父容器），它就像是一个包含了固定定位的后代元素的容器块。

<transform-list> = <transform-function>+

2D Transform Functions

matrix() = matrix( <number> [, <number> ]{5,5} )
specifies a 2D transformation in the form of a transformation matrix of the six values a-f.
translate() = translate( <length> | <percentage> [, <length> | <percentage> ]? )
specifies a 2D translation by the vector [tx, ty], where tx is the first translation-value parameter and ty is the optional second translation-value parameter. If <ty> is not provided, ty has zero as a value.
translateX() = translateX( <length> | <percentage> )
specifies a translation by the given amount in the X direction.
translateY() = translateY( <length> | <percentage> )
specifies a translation by the given amount in the Y direction.
scale() = scale( <number> [, <number> ]? )
specifies a 2D scale operation by the [sx,sy] scaling vector described by the 2 parameters. If the second parameter is not provided, it takes a value equal to the first. For example, scale(1, 1) would leave an element unchanged, while scale(2, 2) would cause it to appear twice as long in both the X and Y axes, or four times its typical geometric size.
scaleX() = scaleX( <number> )
specifies a 2D scale operation using the [sx,1] scaling vector, where sx is given as the parameter.
scaleY() = scaleY( <number> )
specifies a 2D scale operation using the [1,sy] scaling vector, where sy is given as the parameter.
rotate() = rotate( <angle> )
specifies a 2D rotation by the angle specified in the parameter about the origin of the element, as defined by the transform-origin property. For example, rotate(90deg) would cause elements to appear rotated one-quarter of a turn in the clockwise direction.
skew() = skew( <angle> [, <angle> ]? )
specifies a 2D skew by [ax,ay] for X and Y. If the second parameter is not provided, it has a zero value.

Note that the behavior of skew() is different from multiplying skewX() with skewY(). Implementations must support this function for compatibility with legacy content.

skewX() = skewX( <angle> )
specifies a 2D skew transformation along the X axis by the given angle.
skewY() = skewY( <angle> )
specifies a 2D skew transformation along the Y axis by the given angle.

3D Transform Functions

matrix3d() = matrix3d( <number> [, <number> ]{15,15} )
specifies a 3D transformation as a 4x4 homogeneous matrix of 16 values in column-major order.
translate3d() = translate3d( <length> | <percentage> , <length> | <percentage> , <length> )
specifies a 3D translation by the vector [tx,ty,tz], with tx, ty and tz being the first, second and third translation-value parameters respectively.
translateZ() = translateZ( <length> )
specifies a 3D translation by the vector [0,0,tz] with the given amount in the Z direction.
scale3d() = scale3d( <number> , <number>, <number> )
specifies a 3D scale operation by the [sx,sy,sz] scaling vector described by the 3 parameters.
scaleZ() = scaleZ( <number> )
specifies a 3D scale operation using the [1,1,sz] scaling vector, where sz is given as the parameter.
rotate3d() = rotate3d( <number> , <number> , <number> , <angle> )
specifies a 3D rotation by the angle specified in last parameter about the [x,y,z] direction vector described by the first three parameters. A direction vector that cannot be normalized, such as [0,0,0], will cause the rotation to not be applied.

Note that the rotation is clockwise as one looks from the end of the vector toward the origin.

rotateX() = rotateX( <angle> )
same as rotate3d(1, 0, 0, <angle>).
rotateY() = rotateY( <angle> )
same as rotate3d(0, 1, 0, <angle>).
rotateZ() = rotateZ( <angle> )
same as rotate3d(0, 0, 1, <angle>), which is also the same as rotate(<angle>).
perspective() = perspective( <length> )
specifies a perspective projection matrix. This matrix scales points in X and Y based on their Z value, scaling points with positive Z values away from the origin, and those with negative Z values towards the origin. Points on the z=0 plane are unchanged. The parameter represents the distance of the z=0 plane from the viewer. Lower values give a more flattened pyramid and therefore a more pronounced perspective effect. For example, a value of 1000px gives a moderate amount of foreshortening and a value of 200px gives an extreme amount. The value for depth must be greater than zero, otherwise the function is invalid.

rotate(<angle>)，矩形以原点顺时钟旋转，旋转会改变XY轴方向。如，transform: rotate( 45deg )，顺时钟旋转45度。

instance

translate()，控制矩形偏移。取值可以是长度值(如px)和百分比，可以设置两个值(x轴和y轴)。当使用一个值的时候，y轴的偏移默认为0。可以拆分成X轴偏移translateX和Y轴偏移translateY。实际上如果还有rotate()方法，会影响XY轴的方向，就不一定是左移右移了。如，transform: translate( 50px, 50% )，左移50px，下移50%自身高度。

instance

scale()，控制矩形放大或缩小。取值为数字，可以设置两个值(x轴和y轴)。当使用一个值的时候，x和y轴都按这个值的缩放。可拆分成X轴方向放大scaleX和Y轴方向放大scaleY。如，transform: scale( 0.5, 1.5 )，宽度变为原来的0.5倍，高度变为原来的1.5倍。

instance

skew(<angle>)，控制倾斜，可作出菱形。可以设置两个值。当适应一个值的时候，y轴默认取0。可拆分成Y轴向X轴倾斜skewX()，X轴向Y轴倾斜skewY()。如：transform: skew( 30deg, 10deg )，Y轴向X轴倾斜30度，X轴向Y轴倾斜10度。

instance

使用多个方法时，先后顺序不同得到的结果不一样。比如：transform:translate(50%) rotate(45deg)其中的两个方法颠倒后的效果不一样，因为rotate会改变x和y轴的方向。

instance
instance

translate3d()，3D偏移。用法跟translate()方法相同，只不过多了一个z轴的偏移。可以拆分为三个轴方向上各自的方法translateX( <translation-value> )，translateY( <translation-value> )，translateZ( <length> )，z轴的偏移只能用长度值(如px)。如，transform: translate3d( 10px, 20%, 50px );X轴方向移动10px，Y轴移动20%图形高度的距离，向观察者移动50px距离。使用时z轴效果时需要设置透视perspective。

instance

scale3d()，3D缩放。类似scale()方法。可以拆分为三个轴方向上各自的方法scaleX( <number> )X轴方向缩放，scaleY( <number> )Y轴方向缩放，scaleZ( <number> )Z轴方向缩放。使用时z轴效果时需要设置透视perspective。

instance

rotate3d()，3D旋转。类似rotate()方法。可以拆分为三个轴方向上各自的方法rotateX( <angle> )以x轴为中心轴旋转某个角度，rotateY( <angle> )以y轴为中心轴旋转某个角度，rotateZ( <angle> )以z轴为中心轴旋转某个角度。使用时z轴效果时需要设置透视perspective。

transform: rotate3d( 1, 0, 0, 45deg);以x轴上从原点0到1连线为中心轴，即x轴为中心轴旋转45度。

instance

transform: rotate3d( 0, 1, 0, 45deg);以y轴为中心轴旋转45度。

instance

transform: rotate3d( 0, 0, 1, 45deg);以z轴为中心轴旋转45度，相当于2d旋转45度。

instance

transform: rotate3d( 1, 1, 1, 45deg);以原点0到空间坐标为(1, 1, 1)的点的连线为中心轴，旋转45度。

instance

变形坐标轴心点位置|transform-origin

Name: transform-origin
Value:   [ left | center | right | top | bottom | <percentage> | <length> ]
|
  [ left | center | right | <percentage> | <length> ]
  [ top | center | bottom | <percentage> | <length> ] <length>?
|
  [[ center | left | right ] && [ center | top | bottom ]] <length>?
Initial: 50% 50%
Applies to: transformable elements
Inherited: no
Percentages: refer to the size of reference box
Media: visual
Computed value: For <length> the absolute value, otherwise a percentage
Animatable: as simple list of length, percentage, or calc

这个属性是设置在有变形效果的元素上的。可以设置三个值，第三个值是中心点在z轴山的移动。可以设置两个值，对应x轴和y轴，注意会随旋转而改变两个轴的方向。可以设置一个值，但是只是x轴的值，y轴就会默认设为50%。使用长度值，都是相对图形的左上角位置的。或者使用位置关键字。

初始值：50% 50%，即中心点。就像图形中心点被钉住然后顺时钟旋转。

instance
instance

transform-origin:100%; ，X轴上图片的100%位置，Y轴上图片的50%位置为原点。

instance
instance

transform-origin:0 0; 以图片的左上角为原点。就像是左上角被钉住然后顺时钟旋转。

透视度|perspective

Name: perspective
Value: none | <length>
Initial: none
Applies to: transformable elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: Absolute length or "none".
Animatable: as length

这个属性是设置在有变形效果的元素的父容器上的。这里的<length>，表示图像离画布上投影中心的距离。必须使用正数。，可以认为是人眼离图片的距离，看到的效果。与2D变形transform的rotate()方法只是顺时钟旋转不同，perspective属性配合一个3Dtransform方法可以使图像更有立体感。

none，默认，不透视

instance

perspective: 2000px;

instance

perspective: 500px;，越小偏移越明显

instance

透视焦点|perspective-origin

Name: perspective-origin
Value: <position>
Initial: 50% 50%
Applies to: transformable elements
Inherited: no
Percentages: refer to the size of the reference box
Media: visual
Computed value: For <length> the absolute value, otherwise a percentage.
Animatable: as simple list of length, percentage, or calc

这个属性是设置在有变形效果的元素的父容器上的。这个属性设置perspective属性轴心的位置，即视角定点位置。这里<position>指的是位置，可以是位置关键字(如：left)、长度值(如：px)或百分比。所有值都是相对图形的左上角位置的。

perspective-origin: 50% 50%;，默认值，以正中间透视

instance

perspective-origin: left bottom;，以左下角为视角定点位置

instance

perspective-origin: 50% -800px;，图片的中间偏上800px开始透视

instance

perspective-origin: right;，图片右边中间位置开始透视

instance

空间透视|transform-style

Name: transform-style
Value: flat | preserve-3d
Initial: flat
Applies to: transformable elements
Inherited: no
Media: visual
Computed value: Same as specified value.
Percentages: N/A
Animatable: no

这个属性是设置在有变形效果的元素上的。此属性控制父元素内的子元素的透视效果。

flat，初始值，子元素的变形效果在有变形效果的父元素里是扁平的，没有显现出透视效果。

instance

preserve-3d，让一个有变形效果的父元素的子元素的变形效果也显现出来，显现除了变形效果。

instance

并且设置preserve-3d属性值的元素如果有非变形的父元素会有穿透背景的效果

instance

注意：一些CSS属性会在设置到父元素上时，会强制产生平坦的子元素，而不是有3D效果的样子，当这些CSS属性设置到父元素时，会忽略父元素的preserve-3d属性值效果。如：非visible值的overflow，非none值的filter，非auto值的clip，非none值的clip-path，取isolate值的isolation，非none值的mask-image，非none值的mask-boximage-source，非normal值的mix-blend-mode。他们都会使得transform-style属性值设置的效果失效。
