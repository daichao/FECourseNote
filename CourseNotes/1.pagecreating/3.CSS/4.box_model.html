宽度|width

Name: width
Value: <length> | <percentage> | auto
Initial: auto
Applies to: all elements but non-replaced inline elements, table rows, and row groups
Inherited: no
Percentages: refer to width of containing block
Media: visual
Computed value: the percentage or 'auto' as specified or the absolute length; 'auto' if the property does not apply

auto 的真实含义：对于行内元素在无内容时宽度则为0，有内容时则为内容区宽度；对于块级元素，设置 auto 要考虑多重因素： box-sizing 、 border 和 padding 。

  1. 在该元素的 box-sizing 取初始值 content-box 的情况下：
    1. 如果未设置内边距和边框，元素的宽度会达到父元素/根元素的内容区宽度。
    2. 如果设置了内边距和边框，元素的宽度则要用父元素/根元素的内容区宽度减去左右内边距和左右边框的厚度来得到。
  2. 在该元素的 box-sizing 取 border-box 的情况下，不论设不设置内边距和边框，元素的宽度都是等于父元素/根元素的内容区宽度。

一个元素以百分比计算时，以父容器的宽为基数。要跟 auto 区分， auto 是不会超出父元素边框的，而设置百分比是会超出的。如果父容器的宽度取决于这个子元素的宽度（也就是被子元素撑起来的），则得不到正确宽度。

高度|height

Name: height
Value: <length> | <percentage> | auto
Initial: auto
Applies to: all elements but non-replaced inline elements, table columns, and column groups
Inherited: no
Percentages: see prose
Media: visual
Computed value: the percentage or 'auto' (see prose under <percentage>) or the absolute length; 'auto' if the property does not apply

auto 的真实含义：对于行内元素，如果有内容高度为内容区高度，如果没内容高度为当前字体大小撑起的内容区；对于块级元素，设置 auto 要考虑多重因素： box-sizing 、 border 和 padding 。

  1. 在 box-sizing 取初始值 content-box 的情况下:
    1. 当无内容时，高度为0；
    2. 当有内容时，高度为内容区高度。
  2. 在 box-sizing 取 border-box 的情况下:
    1. 当无内容时，
      1. 如果未设置边框和内边距则高度为0；
      2. 如果设置了边框和内边距则高度为上下边框厚度与垂直内边距的和。 height = border-top-width + padding-top + padding-bottom + border-bottom-width
    2. 当有内容时，
      1. 如果未设置边框和内边距则高度为内容区高度；
      2. 如果设置了边框和内边距则高度为内容区高度、上下边框厚度和垂直内边距三项的总和。 height = border-top-width + padding-top + 内容区高度 + border-bottom-width + padding-bottom

Value: <length> | <percentage> | none
Initial: none
Applies to: all elements but non-replaced inline elements, table rows, and row groups
Inherited: no

最大宽高的 none 属性表示对于宽高都没限定。

最大、最小宽高都可以设置百分比，计算时以容器的对应宽高为基数。

内边距|padding

Name: padding
Value: [ <length> | <percentage> ]{1,4}
Initial: 0
Applies to: all elements
Inherited: no
Percentages: width* of containing block
Media: visual
Computed value: <length>
*) if the containing block ih horizontal, otherwise the height

需要注意的是内边距和行高的区别。大部分时候行高撑起了内容区，如果没有个别内容被vertical-align或字体大小提升高度，行高就是内容高度，内容宽度则根据文本多少决定。内边距则是边框以内，内容区以外的部分。其他详情看细节要点分析。

简写边框|border

Name: border
Value: <line-width> || <line-style> || <color>
Initial: See individual properties
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: see individual properties
Animatable: see individual properties

border的每个方向的单独属性还有border-width / border-style / border-color三个单独属性。

border-width : [ <length> | thin | medium | thick ]{1,4}。初始值是 medium 。

border-style : [ none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset ]{1,4}。初始值是 none 。

border-color : [ <color> | transparent ]{1,4}。初始值是当前元素字体颜色 color 。

虽说以上三个属性都有默认值，但是只要 border-style 是默认的 none ，边框就不会出现，也就是不存在颜色和厚度了。

盒子阴影|box-shadow

Name: box-shadow
Value: none | <shadow> [ , <shadow> ]*
Initial: none
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: any <length> made absolute; any specified color computed; otherwise as specified
Animatable: as shadow list

ie8及以下不支持。这个属性用于设置盒子的阴影。第一个值设置水平偏移，第二个值设置垂直偏移，第三个值是模糊半径（可省略，为0），第四个值是阴影大小（可省略，为0），第五个值是颜色（可省略，默认是当前字体颜色）

inset 是设置内阴影。不设置这一项时，就是外阴影。可以叠加多个阴影效果，用英文逗号分隔。阴影不占空间，只作为修饰效果。

外描边|outline

Name: outline
Value: [ <outline-color> || <outline-style> || < outline-width> ]
Initial: see individual properties
Applies to: all elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: see individual properties
Animatable: see individual properties

ie7及以下不支持。设置这个属性并不会占据空间。

这个属性跟border的width、style和color设置相似。只不过不能单独设置四个边。并且多了一个invert来显示与背景色相反的颜色。

宽高设置区域|box-sizing

Name: box-sizing
Value: content-box | border-box
Initial: content-box
Applies to: all elements that accept width or height
Inherited: no
Percentages: N/A
Media: visual
Computed value: specified value
Animatable: no

ie7及以下不支持。这个属性用于指定 width和height 设置的区域。

由于 box-sizing 的初始值是 content-box ，所以一般一个元素的width和height都是设置的的内容区的宽高。

border-box，则是让 width和height 设置的是包括内容区、内边距和边框在内的所有宽高。

盒模型

对于行内元素而言，高度就是行高构成的内容区高度，宽度就是内容区宽度。

而对于块级元素而言，宽高要看box-sizing是设置的content-box还是border-box。设置的是前者着宽高就是内容区宽高，设置的是后者则宽高都要加上边框厚度和内边距。

TRBL

margin / border / padding 都有上右下左四个方向的单独属性。而 border-radius 则是从上左顺时钟走四个角。

撑开盒子

内容区宽/高固定，padding 撑开盒子宽/高。仅对的块级元素。 margin 并不会把父元素容器撑开。

内容覆盖

行内非替换元素 border / padding 设置因为无法像块级元素那样影响垂直高度，所以会使得下面行的内容盖在该元素上，该元素的内容和背景都会覆盖在它的上面一行。但是水平方向仍会像块级元素一样有“撑开”效果。

行高的影响

对于行内非替换元素，设置 margin / border / padding 都不会影响行高。
