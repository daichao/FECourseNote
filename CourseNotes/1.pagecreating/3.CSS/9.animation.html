动画效果不同于过渡效果，它不需要事件触发（如:hover），而是可以在页面加载后就开始自己动起来，并且还可以设置多个关键帧来实现复杂的动画效果。

关键帧|@keyframes

@keyframes关键帧CSS规则。用于控制在动画期间的不同点的CSS动画的属性的值的变化。一个关键帧可以确定在一个循环内的动画效果，动画可以设置循环多次。

使用关键帧实现动画效果时，必须由开发者定义一个动画名称，对应到某个选择器的animation-name属性的属性值，设置关键帧规则@keyframes必须后面跟随开发者定义的动画名称以及每一帧的属性变化必须用两个花括号包裹起来。

可以使用from和to两个关键字或者百分比，表示动画的开始帧到结束帧的变化。如：

div {
  animation-name: slide-right;
  animation-duration: 2s;
}

@keyframes slide-right {
  from {
    margin-left: 0px;
  }
  50% {
    margin-left: 110px;
    opacity: 1;
  }
  50% {
     opacity: 0.9;
  }
  to {
    margin-left: 200px;
  }
}

也可以是纯百分比的，这里要注意开始的是，第一关键帧50%之前都会以50%中定义的属性值来算

@keyframes slide-right {
  50% {
    margin-left: 110px;
    opacity: 0.9;
  }
  100% {
    margin-left: 200px;
  }
}

也可以类似选择器列表的方式

@keyframes flash{
    0%, 50%, 100% {opacity: 1;}
    25%, 75% {opacity: 0;}
}

动画名称|animation-name

Name: animation-name
Value: <single-animation-name>#
Initial: none
Applies to: all elements, ::before and ::after pseudo-elements
Inherited: none
Percentages: N/A
Media: interactive
Computed value: As specified
Animatable: no
Canonical order: per grammar 
<single-animation-name> = none | <IDENT>

定义一个一套动画效果，与@keyframes配合使用。可以使用多个动画效果的动画名称，以逗号隔开。

初始值none，清除某个元素的所有动画效果，一旦设置这个属性值，所有动画效果无效。

instance

其他属性值由开发者自定义，与@keyframe配合使用

动画方向及往返|animation-direction

Name: animation-direction
Value: <single-animation-direction>#
Initial: normal
Applies to: all elements, ::before and ::after pseudo-elements
Inherited: no
Percentages: N/A
Media: interactive
Computed value: As specified
Animatable: no
Canonical order: per grammar 
<single-animation-direction> = normal | reverse | alternate | alternate-reverse

这个属性定义了动画在所有循环期间的播放方向。需要注意的是，循环是从1开始计数的，并且往返<animation-iteration-count>都计次数的，所以正方向去一次算一次，返回来一次也算一次，所以才有奇偶的说法。

初始值normal，动画在所有循环期间按照原来指定的方向播放

instance

reverse，动画在所有循环期间按照原来指定方向的反方向播放

instance

alternate，动画在奇数循环时以原来指定的方向播放，在偶数循环时以原来方向的反方向播放

instance

alternate-reverse，跟alternate相反，动画在偶数循环时以原来指定的方向播放，在奇数循环时以原来方向的反方向播放

instance

该属性定义动画开始的时候是否保持第一帧或最后帧的状态。

初始值none，不做设置

forwards，结束的时候保持动画在最后一帧的状态

简写动画属性|animation

Name: animation
Value: <single-animation>#
Initial: see individual properties
Applies to: all elements, ::before and ::after pseudo-elements
Inherited: no
Percentages: N/A
Media: interactive
Computed value: As specified
Animatable: no
Canonical order: per grammar 
<single-animation> = <single-animation-name> || <time> || <single-animation-timing-function> || <time> ||
                     <single-animation-iteration-count> || <single-animation-direction> || <single-animation-fill-mode> || <single-animation-play-state>

以上八个属性的简写形式。类似于<transition>整个属性值表的书写有顺序要求。首先，两个时间有先后顺序要求，第一个是<animation-duration>，第二个是<animation-delay>。其次，动画名称<animation-name>必须放在第一个属性值的位置，否则会被浏览器错误的解析为CSS属性值，进而产生错误。多个动画效果（多个属性值列表）可以用逗号隔开一同声明，但是要注意的是，如果有一个动画效果的定义是none则整个声明失效。

初始值none，不做动画。如果设置这个属性值，则清除所有动画效果。

animation: moves 2s ease 0s 1 normal none running;

instance

实现两个关键帧同时使用

@keyframes changecolor {
            from {
                background: rgba(0, 255, 0, 0.7);
            }
            50% {
                background: rgba(0, 255, 255, 0.5);
                height: 80px;
            }
            to {
                background: rgba(255, 0, 255, 0.7);
                height: 100px;
            }
}
@keyframes moves {
            0% {
                left: 0;
                top: 0;
            }
            50% {
                left: 250px;
                top: 100px;
            }
            100% {
                left: 500px;
                top: 0;
            }
}

#compound+pre+.example>div {
    animation: moves 3s cubic-bezier(0, 0.1, 0.7, 1) 0s infinite alternate both running, changecolor ease 1s infinite alternate-reverse forwards paused;
}
