属性过渡|transition-property

Name: transition-property
Value: none | <single-transition-property>#
Initial: all
Applies to: all elements, ::before and ::after pseudo elements
Inherited: no
Percentages: N/A
Media: visual
Computed value: Same as specified value.
Animatable: no
Canonical order: per grammar
<single-transition-property> = all | <custom-ident>;

这个属性定义了一个元素的某些CSS属性发生变化时是否有过渡效果。一般是跟事件触发或者CSS的:hover伪类一起使用。通常<custom-ident>指的就是事件触发时改变的某个CSS属性名。注意：一定要有过渡时transition-duration属性长才能看得出来。

transition-property: none;没有CSS属性会发生过渡效果

none

transition-property: all;，默认，对于所有发生变化的效果，都设置过渡效果

all

transition-property: left;只对left属性设置的变化，设置过度效果

left

transition-property: left, color;只对color属性设置和left属性设置的变化，设置过渡效果

left,
color

注意：开启属性过渡效果时，必须给需要过渡的属性设置某个初始值，以配合事件触发时改变的终值平滑过渡，否则某些浏览器未默认设置初始值的属性，在过渡时会有闪烁不稳定现象。

cubic-bezier(<number>, <number>, <number>, <number>)。cubic-bezier(x1, y1, x2, y2)，四个值分别表示两个点的横纵坐标，必须是数字。第一个和第三个是两个点的x轴坐标，必须是0~1之间的正数。

y轴坐标可以为负数，可以大于1。这两个坐标点对应到一种速率曲线，横坐标表示时间总的时间比例（满值为1），纵坐标表示对应CSS属性变化的比例，两个坐标分别表示开始过渡时的CSS属性值和结束过渡时的CSS属性值的比例。如果向左100px的位移过渡，则设置y2大于1的话，结束动画时元素会从目标位置超出一定比例距离，最后再回到目标位置。y轴坐标负值表示CSS属性值变化会先按照变化的反方向变化一定比例，最后回到目标属性值，而相应点的x轴坐标则是表示y轴坐标CSS属性值变化发生时占整个动画的时间比例中的某个点。

简写属性过渡|transition

Name: transition
Value: <single-transition>#
Initial: see individual properties
Applies to: all elements, ::before and ::after pseudo elements
Inherited: no
Percentages: N/A
Media: interactive
Computed value: see individual properties
Animatable: no
Canonical order: per grammar
<single-transition> = [ none | <single-transition-property> ] || <time> || <single-transition-timing-function> || <time>

简写属性过渡就是把前面四个过渡属性合并在一起。要注意先后顺序，第一个时间会被浏览器解析为<transition-duration>，第二个时间则是<transition-delay>。如果只有一个时间则只表示<transition-duration>。多个过渡效果以逗号隔开，但是如果其中有一个声明是none，则整个声明失效。

示例：transition: left 2s ease 1s, color 2s;
